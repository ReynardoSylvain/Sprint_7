# Sprint_7 Яндекс.Практикум

## Описание структуры проекта

Этот проект представляет собой набор тестов для API сервиса Яндекс.Самокат. Он включает в себя проверку создания курьера, авторизации, создания заказа и получения списка заказов.

### 📁 Корень проекта

#### Файлы

*   `README.md` - Описание структуры проекта
*   `requirements.txt` — Список всех модулей и пакетов Python, необходимых для работы проекта.

### 📁 data - данные для работы проекта
* `api_client.py` - Класс для отправки HTTP-запросов к API.
* `utils.py` - Вспомогательные функции для генерации случайных строк.
* `urls.py` - Файл, содержащий базовый URL для API.

### 📁 tests - директория с тестами

*   `courier_tests.py` - Тесты для API создания курьеров.
*   `courier_login_tests.py` - Тесты для API авторизации курьеров.
*   `order_tests.py` - Тесты для API создания заказов.
*   `get_orders_list_tests.py` - Тесты для API получения списка заказов.

### 📁 allure-results - директория с отчетами о тестировании Allure

*   Содержит файлы отчетов о тестировании от фреймворка Allure.

## Запуск тестов

Для запуска тестов необходимо:

1. Убедиться, что у вас установлены все зависимости, перечисленные в `requirements.txt` (`pip install -r requirements.txt`).
2. Перейти в каталог проекта.
3.  Запустить тесты с помощью команды:

    ```bash
    pytest -v --alluredir=allure-results tests/
    ```
4. Для просмотра отчёта Allure нужно выполнить команду
    ```bash
    allure serve allure-results
    ```

## Описание проверок

*   **Создание курьера:** Проверка создания курьера, дубликатов и обязательных полей.
*   **Логин курьера:** Проверка авторизации курьера и разных вариантов ошибок.
*   **Создание заказа:** Проверка создания заказа с разными цветами.
*   **Получение списка заказов:** Проверка получения списка заказов от API.
